# Common HTML Tags

HTML (HyperText Markup Language) is the standard markup language for creating web pages. Here are some of the most common HTML tags:

## Document Structure

*   `<!DOCTYPE html>`: Defines the document type to be HTML5.
*   `<html>`: The root element of an HTML page.
*   `<head>`: Contains meta-information about the HTML page, such as the title.
*   `<title>`: Specifies the title of the page, which is shown in the browser's title bar or tab.
*   `<body>`: Contains the visible page content.

## Text Content

*   `<h1>` to `<h6>`: Headings, with `<h1>` being the most important.
*   `<p>`: A paragraph.
*   `<a>`: A hyperlink.
*   `<strong>`: Important text (usually displayed as bold).
*   `<em>`: Emphasized text (usually displayed as italic).
*   `<span>`: An inline container used to mark up a part of a text, or a part of a document.

## Lists

*   `<ul>`: An unordered list.
*   `<ol>`: An ordered list.
*   `<li>`: A list item.

## Media

*   `<img>`: An image.
*   `<video>`: A video.
*   `<audio>`: Audio content.

## Forms

*   `<form>`: An HTML form for user input.
*   `<input>`: An input field.
*   `<textarea>`: A multi-line input control.
*   `<button>`: A clickable button.
*   `<label>`: A label for an `<input>` element.

## Tables

*   `<table>`: A table.
*   `<tr>`: A table row.
*   `<th>`: A table header cell.
*   `<td>`: A table data cell.
