# CSS Typography

Typography is the art of arranging text to make it legible, readable, and appealing when displayed.

## Common Typography Properties

*   `font-family`: Specifies the font for an element. You can provide a list of fonts, and the browser will use the first one that is available.
    ```css
    p {
      font-family: "Helvetica Neue", Arial, sans-serif;
    }
    ```
*   `font-size`: Sets the size of the font.
    ```css
    h1 {
      font-size: 2.5rem; /* 2.5 times the root font size */
    }
    ```
*   `font-weight`: Sets the thickness of the font.
    ```css
    .bold {
      font-weight: bold;
    }
    ```
*   `font-style`: Sets the style of the font (e.g., `normal`, `italic`).
*   `text-align`: Aligns the text within an element (`left`, `right`, `center`, `justify`).
*   `line-height`: Sets the distance between lines of text.
*   `color`: Sets the color of the text.
*   `text-decoration`: Adds decoration to text (e.g., `underline`, `line-through`).
*   `text-transform`: Changes the case of the text (e.g., `uppercase`, `lowercase`, `capitalize`).
